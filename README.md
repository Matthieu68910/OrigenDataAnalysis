# OrigenDataAnalysis

## Overview
OrigenDataAnalysis is a Python-based assistance program designed to facilitate the analysis of data generated by the Origen22 program. This tool aims to streamline data processing, offering users an efficient and user-friendly experience in handling complex datasets.

## Features
- **Data Import**: Easily import data files from Origen22.
- **Data Analysis**: Perform comprehensive data analysis with built-in functions.
- **Visualization Tools**: Generate graphs and charts to visualize the analyzed data.
- **Export Options**: Export the processed data and visuals in various formats.

## Getting Started

### Prerequisites
- Python 3.x
- Relevant Python libraries

### Installation
1. Clone the repository: `git clone https://github.com/Matthieu68910/OrigenDataAnalysis.git`
2. Install the required Python libraries: `pip install -r requirements.txt`
