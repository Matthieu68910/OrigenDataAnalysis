/_!_\  WARNING  Only for use on certain type of output from Origen22! /_!_\  

See examples of supported data files on github : https://github.com/Matthieu68910/OrigenDataAnalysis.git

Lauch: Click on the executable [- OrigenDA.exe -] to start the program

Analyse file:
	- Click [- Open File -] and select data file in the browser, or enter path manually
	- Click [- Process file -]. Check the success of the analysis on the right, and get informations about content
	- Under "Plot Settings", select the desired element, then an isotope and the origin of it.
	- If you want, select the type of axis (default to 'linear')
	- Click [- Process plot -]

Save figure:
	- After choosing the right isotope and origin, choose a "Name", "Resolution", "Width" and "Height".
	- Select the type of axis.
	- If needed, click [- Show Plot -]
	- Click [- Save Plot -]

Saved Figure: you can find all the saved graphics in the folder /OrigenDA/Fig/
